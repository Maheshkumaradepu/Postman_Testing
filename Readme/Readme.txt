To Create dummy api  we need nodejs & json server
1.Install nodejs 
2.open cmd promt and check if nodejs is installed or not
  #node--version
  #npm--version
3.Install json server form terminal/cmd promt
  #npm install -g json-server
4.Now create a .json file which has data in json format and open cmd promt from same location and run below command to create the 
   server for the same file

  #json-server students.json
      /* this will start the server and never close the Cmd after until you want to stop the Server */


5.Now you can use the location or server link to trigger the apis from Postman.
